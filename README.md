# Система управления строительными проектами `Smradity`

---
## Подготовка
Создаем в корне проекта файл `.env` и указваем параметры среды
```environment
SERVER_PORT=9090
```
---
## Развертывание

1. Сборка проекта
```bash
make build
```
2. Запуск
```bash
make run
```
3. Остановка
```bash
make stop
```
4. Просмотреть логи
```bash
make logs
```
